FutsalMind WebApp

Guida completa: setup in locale e deploy gratuito su Vercel

REACT · VITE · SUPABASE · VERCEL
⚙️ Parte 1 — Avvio in Locale
1Posiziona i file del progetto

Crea una cartella sul tuo PC, ad esempio C:\Progetti\FutsalMind-Web, e copia dentro tutti i file del progetto web (da questo ZIP).

📁 Struttura attesasrc/ · index.html · package.json · vite.config.ts · vercel.json · .env.example
2Crea il file .env

Copia il file .env.example e rinominalo in .env. Poi apri il file e inserisci le tue chiavi Supabase (le stesse che hai già usato nel progetto mobile).

VITE_SUPABASE_URL=https://tuoprogettoXXXXX.supabase.co
VITE_SUPABASE_ANON_KEY=eyJhbGciOi...
⚠️ NON caricare il file .env su GitHub!Contiene le chiavi private della tua app. Aggiungilo a .gitignore.
3Installa le dipendenze e avvia

Apri il Terminale nella cartella del progetto:

C:\Progetti\FutsalMind-Web> npm install
C:\Progetti\FutsalMind-Web> npm run dev

Apri il browser su http://localhost:5173 — vedrai FutsalMind!

✅ Funziona su mobile anche in locale!Nella stessa rete WiFi, apri sul telefono l'indirizzo IP che mostra Vite nel terminale, es. http://192.168.1.xx:5173
🚀 Parte 2 — Deploy su Vercel (Online Gratis)
4Crea account GitHub e carica il codice

Vercel funziona meglio connesso a GitHub. Se non ce l'hai, crea un account gratuito su github.com.

Installa Git su Windows da git-scm.com, poi:

C:\Progetti\FutsalMind-Web> git init
C:\Progetti\FutsalMind-Web> echo ".env" >> .gitignore
C:\Progetti\FutsalMind-Web> git add .
C:\Progetti\FutsalMind-Web> git commit -m "FutsalMind webapp"

Su GitHub, crea un nuovo repository chiamato futsalmind-web (privato o pubblico), poi:

C:\Progetti\FutsalMind-Web> git remote add origin https://github.com/TUOUSERNAME/futsalmind-web.git
C:\Progetti\FutsalMind-Web> git push -u origin main
5Connetti Vercel al repository
  • Vai su vercel.com e crea account gratuito (con GitHub)
  • Clicca "Add New Project"
  • Seleziona il repository futsalmind-web
  • Framework Preset: scegli "Vite"
  • Non modificare Build Command e Output Directory (già corretti)
⚠️ NON cliccare ancora "Deploy"!Prima devi aggiungere le variabili d'ambiente nel passo successivo.
6Aggiungi le variabili d'ambiente su Vercel

Nella schermata di import progetto su Vercel, espandi la sezione "Environment Variables" e aggiungi:

  • Nome: VITE_SUPABASE_URL — Valore: il tuo URL Supabase
  • Nome: VITE_SUPABASE_ANON_KEY — Valore: la tua chiave anon Supabase
ℹ️ Dove trovo le chiavi?Su supabase.com → il tuo progetto → Settings → API
7Deploy! 🚀

Clicca "Deploy". Vercel costruirà l'app in circa 1 minuto.

Al termine riceverai un URL del tipo: futsalmind-web.vercel.app

🎉 La tua webapp è online!Accessibile da qualsiasi browser, su PC, telefono e tablet. Ogni volta che fai git push, Vercel rideploya automaticamente.
8Configura Supabase per accettare il dominio

Su supabase.com → il tuo progetto → Authentication → URL Configuration, aggiungi:

  • Site URL: https://futsalmind-web.vercel.app
  • Redirect URLs: https://futsalmind-web.vercel.app/**

Senza questo, il login via email non funzionerà correttamente in produzione.

🔧 Comandi Utili
▶Riferimento rapido comandi
  • npm run dev — Avvia in locale (sviluppo)
  • npm run build — Costruisce la versione di produzione nella cartella dist/
  • npm run preview — Anteprima locale della build di produzione
  • git push — Carica su GitHub → Vercel rideploya automaticamente
